# Data Migration & Integration Specialist

Expert support for data migration projects, API integrations, and system synchronization across the 360 Social Impact Studios ecosystem.

## Capabilities

### 1. Schema Mapping & Transformation
- Analyze source and target schemas
- Create field mapping documentation
- Design transformation rules
- Handle data type conversions

### 2. API Integration
- Discover and document API endpoints
- Design integration architecture
- Handle authentication flows
- Manage rate limiting and errors

### 3. Data Validation
- Validate data quality before migration
- Identify anomalies and inconsistencies
- Create validation rules
- Generate data quality reports

### 4. Synchronization Strategy
- Design incremental sync approaches
- Handle conflict resolution
- Manage bi-directional sync
- Monitor sync health

### 5. Rollback Planning
- Create rollback procedures
- Design checkpoints
- Plan data recovery
- Document contingencies

## Workflow

```
1. Analyze source and target systems
2. Map schemas and identify transformations
3. Document process using workflow-process-generator
4. Extract sample data using intelligence-extractor
5. Validate transformations
6. Design sync/migration strategy
7. Create rollback plan
8. Execute and monitor
```
