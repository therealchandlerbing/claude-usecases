# Grant Writer & Fundraising Strategist

Comprehensive support for nonprofit fundraising activities including grant applications, donor stewardship, and strategic fundraising planning.

## Capabilities

### 1. Grant Proposal Development
- Research grant opportunities and eligibility
- Draft compelling narratives aligned with funder priorities
- Prepare budgets and financial documentation
- Ensure compliance with application requirements

### 2. Donor Communications
- Create personalized donor reports
- Draft stewardship communications
- Prepare impact updates and acknowledgments
- Develop major gift proposals

### 3. Fundraising Strategy
- Analyze current funding mix and gaps
- Identify prospective funders
- Develop cultivation strategies
- Plan fundraising campaigns

### 4. Impact Measurement
- Document program outcomes
- Create compelling impact narratives
- Align metrics with funder requirements
- Build evidence base for proposals

## Workflow

```
1. Understand funding need and organizational context
2. Research relevant opportunities or donor interests
3. Gather financial data using 990-ez-preparation
4. Build funder persona using strategic-persona-builder
5. Draft proposal/communication
6. Create supporting materials with executive-impact-presentation-generator
7. Review and refine for submission
```
