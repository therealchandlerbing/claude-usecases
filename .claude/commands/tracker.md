# Create Project Status Tracker

Build a multi-source project tracker using the **360 Executive Project Tracker** skill.

## Instructions

Invoke the `360-executive-project-tracker` skill to consolidate project information from multiple sources into Excel dashboards and interactive HTML visualizations.

## Data Sources

The tracker pulls from:
- **Gmail** - Status updates, blocker mentions, task assignments
- **Google Calendar** - Meeting notes, action items, follow-ups
- **Google Drive** - Meeting deliverables from multiple folders
- **Asana** - Direct project and task integration

## What Gets Generated

**Excel Tracker (Source of Truth):**
- Professional 16-column structure with formulas
- Conditional formatting for visual status
- Dashboard sheet with real-time metrics
- Category-based workstream organization
- Blocker alerts (warning at 3 days, critical at 7+)

**HTML Dashboard (Executive Presentation):**
- Interactive filtering by status, priority, owner, category
- Real-time search across all fields
- Visual progress indicators
- Blocker and risk highlighting

## Use Cases

- Comprehensive project status tracking
- Multi-project portfolio monitoring
- Executive dashboards for visibility
- Blocker detection and escalation
- Team workload and capacity tracking
- Weekly status reporting automation
