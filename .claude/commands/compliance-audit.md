# Compliance & Process Audit

**Workflow Template**: Ethics review + process documentation for compliance readiness

---

## Workflow Steps

Execute the following skills in sequence:

### Step 1: AI Ethics Assessment
**Skill**: `ai-ethics-advisor`

Evaluate AI systems for compliance:
- Bias assessment
- Fairness evaluation
- Transparency review
- Privacy compliance
- EU AI Act / regulatory alignment
- Risk classification

### Step 2: Process Documentation
**Skill**: `workflow-process-generator`

Document current processes:
- Standard Operating Procedures
- Process flowcharts (BPMN)
- RACI matrices
- Decision trees
- Audit trails

### Step 3: Gap Analysis
**Skill**: `skill-orchestrator`

Identify compliance gaps:
- Compare current state to requirements
- Prioritize remediation items
- Risk-rank gaps
- Create action plan

### Step 4: Remediation Planning
**Skill**: `workflow-process-generator`

Create remediation documentation:
- Updated SOPs with controls
- New process requirements
- Training materials
- Change management plan

### Step 5: Audit Package Assembly
**Skill**: `design-director`

Prepare audit-ready materials:
- Professional formatting
- Document version control
- Index and navigation
- Print-ready output

---

## Required Input

Provide:
1. **Audit Scope** - Which systems/processes to review
2. **Regulatory Framework** - EU AI Act, ISO, FDA, SOX, etc.
3. **Current Documentation** - Existing SOPs, policies
4. **AI Systems Inventory** - Systems using AI/ML
5. **Timeline** - Audit date or compliance deadline

---

## Output Deliverables

1. **Ethics Assessment Report** (Markdown)
   - Risk classification by system
   - Bias and fairness findings
   - Compliance gap analysis
   - Remediation recommendations

2. **Process Documentation Package**
   - SOPs for key processes
   - Process maps (Mermaid/BPMN)
   - RACI matrices
   - Decision trees

3. **Gap Analysis Matrix** (Excel)
   - Requirements vs. current state
   - Risk-ranked gaps
   - Remediation priority

4. **Remediation Plan** (Markdown + Timeline)
   - Action items with owners
   - Timeline and milestones
   - Resource requirements

5. **Audit-Ready Binder** (PDF)
   - Complete documentation package
   - Version-controlled
   - Index and cross-references

---

## Execution Instructions

1. Define audit scope and frameworks
2. Run AI ethics assessment on in-scope systems (30-60 min)
3. Document current processes (45-60 min per process)
4. Perform gap analysis (30-45 min)
5. Create remediation plans (30-45 min)
6. Assemble and polish audit package (30-45 min)
7. Review with compliance stakeholders

---

## Compliance Frameworks Supported

- **AI Governance**: EU AI Act, NIST AI RMF, IEEE EAD
- **Quality Systems**: ISO 9001, ISO 13485, ISO 27001
- **Healthcare**: FDA QSR, HIPAA, 21 CFR Part 11
- **Financial**: SOX, PCI-DSS
- **General**: GDPR, CCPA
