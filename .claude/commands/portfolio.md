# Build Client Portfolio Page

Create a professional HTML portfolio page for a 360 client venture using the **360 Client Portfolio Builder** skill.

## Instructions

Invoke the `360-client-portfolio-builder` skill to transform Vianeo business validation sprint data into a production-quality portfolio page.

## Required Input

Provide:
1. **Client/Venture Name**
2. **Vianeo Sprint Data** - Validation results, personas, market analysis
3. **Key Assets** - Logo, images, team photos (or placeholders)
4. **Target Audience** - Investors, partners, ecosystem stakeholders

## What Gets Generated

**Primary Deliverable:**
- Production-quality single-page HTML (150-250KB)
- Self-contained (embedded CSS, JS, fonts via CDN)
- Mobile-responsive (375px to 1920px+)
- Fast loading (under 3 seconds on 4G)
- Deployment-ready (no build process)

**Supporting Deliverables:**
- Content strategy brief
- Design rationale documentation
- Deployment instructions
- Update guide for simple changes

## Success Criteria

- External stakeholders understand value proposition in 10 seconds
- Venture appears professional without overselling traction
- 360's partnership value is clear but doesn't overshadow client
- Clear next steps for investors, partners, collaborators
- Works across all devices and accessibility tools

## Estimated Time

2-3 hours per portfolio page
