# Quarterly Report Package

**Workflow Template**: Comprehensive quarterly reporting for board and stakeholders

---

## Workflow Steps

Execute the following skills in sequence:

### Step 1: Financial Analysis
**Skill**: `financial-modeling-skills`

Compile quarterly financial performance:
- Revenue and expense analysis
- Budget vs. actual comparison
- Cash flow summary
- Key financial metrics
- Forecast adjustments

### Step 2: Impact Measurement
**Skill**: `financial-modeling-skills` (Impact Modeling)

Calculate and document impact:
- SROI calculations
- SDG contribution mapping
- Beneficiary metrics
- Impact narrative

### Step 3: Generate Impact Report
**Skill**: `executive-impact-presentation-generator`

Create dual-format impact reports:
- Presentation deck for board meeting
- Executive document for distribution
- Data visualizations
- Brand-compliant design

### Step 4: Portfolio Performance
**Skill**: `financial-modeling-skills` (Portfolio Intelligence)

Analyze portfolio health:
- Cross-portfolio performance
- Individual venture updates
- Risk assessment
- Strategic allocation review

### Step 5: Board Materials Preparation
**Skill**: `360-board-meeting-prep`

Prepare governance-ready package:
- Financial dashboard
- Portfolio health summary
- Strategic initiatives update
- Motions requiring approval
- Compliance documentation

### Step 6: Design Elevation
**Skill**: `design-director`

Apply professional polish:
- Consistent visual identity
- Publication-quality formatting
- Accessibility compliance
- Print optimization

---

## Required Input

Provide:
1. **Quarter & Year** - e.g., Q4 2024
2. **Financial Data** - Revenue, expenses, cash position
3. **Portfolio Updates** - Status of each venture/project
4. **Impact Data** - Beneficiary numbers, outcomes
5. **Strategic Context** - Key decisions, pivots, achievements

---

## Output Deliverables

1. **Financial Report** (Excel + Summary)
   - P&L, cash flow, balance sheet summary
   - Budget variance analysis
   - Forecast updates

2. **Impact Report** (Deck + Document)
   - Dual-format board-ready report
   - SROI and impact metrics
   - Beneficiary stories

3. **Portfolio Dashboard** (HTML + Excel)
   - Cross-portfolio performance
   - Individual venture scorecards
   - Risk heat map

4. **Board Packet** (DOCX + PDF)
   - Complete governance package
   - Motions and resolutions
   - Compliance attestations

5. **Executive Summary** (1-pager)
   - Quarter highlights
   - Key metrics at a glance
   - Strategic priorities

---

## Execution Instructions

1. Gather all quarterly data
2. Run financial analysis (30-40 min)
3. Calculate impact metrics (20-30 min)
4. Generate impact report (20-30 min)
5. Compile portfolio performance (20-30 min)
6. Prepare board materials (30-45 min)
7. Apply design polish (20-30 min)
8. Final review and distribution prep

---

## Timeline Recommendation

**Start**: 2 weeks before board meeting
**Duration**: 3-4 hours total (can be spread across days)
**Review Cycle**: Allow 3-5 days for leadership review
