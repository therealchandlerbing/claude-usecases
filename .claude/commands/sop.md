# Generate Process Documentation

Create professional workflow documentation using the **Workflow Process Generator** skill.

## Instructions

Invoke the `workflow-process-generator` skill to transform organizational knowledge into compliance-ready documentation.

## Documentation Types

### 1. Standard Operating Procedures (SOPs)
- Step-by-step procedural documentation
- Decision trees and RACI matrices
- Compliance integration and version control

### 2. Process Maps & Flowcharts
- BPMN 2.0 notation
- Swimlane diagrams
- Value stream mapping
- Process hierarchy documentation

### 3. Runbooks & Playbooks
- Incident response procedures
- Troubleshooting decision trees
- Escalation matrices
- Recovery procedures

### 4. Work Instructions & Job Aids
- Task-level procedural documentation
- Quick reference cards
- Decision support tools

### 5. Process Governance Documentation
- Document control procedures
- Change request workflows
- Audit preparation materials
- Process metrics

### 6. Training & Onboarding Materials
- Learning objectives
- Curriculum design
- Assessments and competency verification

## Core Principles

1. **Clarity**: Unambiguous steps executable on first attempt
2. **Compliance-Ready**: ISO, FDA, SOX audit-ready
3. **User-Centered**: Designed for actual users
4. **Continuous Improvement**: Built-in feedback mechanisms
5. **Scalability**: Single-team to enterprise deployment

## Output Features

- Version control and approval workflows
- Audit trails for regulatory compliance
- Visual hierarchy and professional design
- Mermaid diagrams for process flows
- Change management integration
