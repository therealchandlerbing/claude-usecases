# Board Meeting Preparation

Prepare comprehensive board meeting materials using the **360 Board Meeting Prep** skill.

## Instructions

Invoke the `360-board-meeting-prep` skill to generate governance-ready board meeting packets.

## Required Information

Provide:
1. **Meeting Date** - When is the board meeting?
2. **Meeting Type** - Regular quarterly, special session, annual
3. **Key Topics** - Agenda items or focus areas

## 6-Phase Workflow

**Phase 1: Data Collection** (T-10 days)
- Confirm meeting parameters
- Pull data from Asana, Drive, Gmail
- Identify information gaps

**Phase 2: Draft Generation** (T-7 days)
- Financial dashboard
- Portfolio health summary
- Strategic initiatives update
- Governance compliance report
- Motions requiring approval

**Phase 3: Review & Refinement** (T-7 to T-2)
- Incorporate feedback
- Iterative updates
- Final approval from leadership

**Phase 4: Final Output** (T-2 days)
- DOCX document package
- HTML dashboards
- Design-director elevation
- Quality assurance

**Phase 5: Meeting Support** (Day of)
- Pre-meeting readiness check
- Real-time motion tracking
- Action item capture

**Phase 6: Post-Meeting** (T+2 days)
- Meeting minutes
- Action items to Asana
- Document archival

## Output Formats

- DOCX board packet
- Interactive HTML dashboards
- PDF versions for distribution
