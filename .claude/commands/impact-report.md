# Generate Impact Report

Create board-ready impact reports using the **Executive Impact Presentation Generator** skill.

## Instructions

Invoke the `executive-impact-presentation-generator` skill to produce dual-format impact reports (presentation deck + executive document).

## Dual Output Formats

**1. Presentation Deck**
- Visual-first design for meetings
- Key metrics and highlights
- Chart-heavy with minimal text
- Designed for projection/screen sharing

**2. Executive Document**
- Detailed narrative format
- Comprehensive data and analysis
- Print-optimized layout
- Suitable for board packets

## Six-Section Framework

Both formats follow consistent structure:
1. **Executive Summary** - Key achievements and metrics
2. **Strategic Progress** - Goals and milestone tracking
3. **Impact Metrics** - Quantified outcomes
4. **Portfolio/Program Updates** - Detailed status
5. **Financial Summary** - Budget and resource allocation
6. **Forward Look** - Upcoming priorities and risks

## Features

- Brand customization (colors, logos, fonts)
- Accessibility compliance (WCAG AA)
- Print optimization
- Interactive elements (digital version)
- Chart.js visualizations

## Generation Time

10-20 minutes depending on content complexity

## Use Cases

- Quarterly board presentations
- Investor updates
- Annual impact reports
- Stakeholder communications
- Grant reporting
