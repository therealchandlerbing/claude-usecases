# Generate Newsletter or Executive Brief

Create publication-style newsletters using the **360 Newsletter Generator** skill.

## Instructions

Invoke the `360-newsletter-generator` skill to synthesize updates from Asana, Gmail, and Google Drive into professionally designed HTML dashboards.

## Two Output Formats

**1. Executive Brief (Confidential)**
- Audience: Leadership, board, investors
- Content: Financial data, HR updates, strategic decisions
- Trigger: "Generate executive brief" / "Create board update"

**2. General Newsletter (Public/Stakeholder)**
- Audience: Partners, stakeholders, public
- Content: High-level updates, impact stories
- Trigger: "Generate newsletter" / "Weekly digest"

## Data Integration

**Asana Integration (70% weight)**
- Project status and milestones
- Task completions and blockers
- Team assignments and capacity

**Gmail Integration**
- Important communications
- Partnership updates
- Decision threads

**Google Drive Integration**
- Recent document activity
- Meeting notes and action items
- Strategic documents

## Output Features

- British newspaper-inspired dense design
- Interactive Chart.js visualizations
- Print-friendly PDF export
- Executive dashboard aesthetic
- Professional design with interactive components

## When NOT to Use

- Real-time dashboards (this is retrospective)
- Email marketing campaigns
- Press releases
- Social media content
