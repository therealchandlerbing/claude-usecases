# Daily CEO Intelligence Brief

Generate my morning executive brief using the **CEO Advisor** skill.

## Instructions

Invoke the `ceo-advisor` skill to generate a comprehensive daily intelligence brief covering:

1. **Critical Issues** - Any urgent matters requiring immediate attention
2. **Intelligence Scan** - Key signals across internal and external sources
3. **Stakeholder Updates** - Relationship health and engagement recommendations
4. **Performance Metrics** - Team health, project status, financial indicators
5. **Today's Priorities** - Top 3 focus areas for the day

## Output Format

Deliver as an HTML dashboard with:
- Priority alerts at the top (CRITICAL/HIGH/STANDARD/INFO)
- Key metrics visualization
- Actionable recommendations with specific next steps
- Timeline for the day

## Data Sources

Pull from:
- Asana (project status, tasks, blockers)
- Gmail (important communications)
- Calendar (today's meetings)
- Previous conversations (context)

---

**Skill**: ceo-advisor
**Trigger**: "Morning brief" / "Daily intelligence" / "What should I know today?"
