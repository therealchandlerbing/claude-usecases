## Usage Guidelines

### For Team Alignment
- Share full document for comprehensive understanding
- Use Key Quotes section for quick reference
- Reference Evidence Appendix when challenged

### For Decision-Making
- Check Quality Score before relying on claims
- Note validation status per element
- Consider Research Gaps before major decisions

### For External Presentation
- Use Executive Summary for overview
- Select 2-3 Key Quotes per persona
- Highlight Strategic Implications

### For Iteration
- Schedule review at [timeframe]
- Collect new data addressing flagged gaps
- Re-generate with enhanced evidence

## Template Usage Notes

### Evidence Markers
Use these markers throughout to indicate validation status:
- No marker = Validated with evidence
- ◐ = Partially validated (some evidence)
- ⚠️ = Inferred (needs validation)

### Length Guidelines
- Executive Summary: 200-400 words
- Each Layer: 50-150 words per component
- Quality Assessment: 100-200 words per persona
- Total document: 2,000-3,000 words per persona

### Quality Standards
- All Layer sections populated
- Evidence sources documented
- Direct quotes included where available
- Validation status marked
- Actionable recommendations provided
