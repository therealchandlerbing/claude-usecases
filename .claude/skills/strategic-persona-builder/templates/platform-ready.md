## Entry Checklist

Before platform entry, verify:

- [ ] All text fields: 2-3 sentences max
- [ ] All bullet lists: 4-6 items
- [ ] Need statements: Under 60 characters
- [ ] Names: Specific (not generic)
- [ ] Ages: Exact numbers (not ranges)
- [ ] Validation badges: Accurate status
- [ ] No markdown formatting in entry text

## Validation Badges

Use these badges in Quick Reference:

- **✓ Validated** - 5+ interviews with direct quotes
- **◐ Partially Validated** - 3-4 interviews or mixed sources
- **⚠️ Inferred** - <3 interviews or behavioral data only

---

## Character Count Examples

**Good Need Statement:**
"Reduce validation time by 50%" [28/60 chars] ✓

**Too Long:**
"Significantly reduce the amount of time spent on redundant validation activities" [79/60 chars] ✗

## Copy-Paste Notes

When copying to Vianeo platform:

1. Remove all markdown formatting (**, •, ---)
2. Remove character count annotations
3. Remove validation badges from text
4. Keep text in plain format
5. Verify character limits in platform

## Constraint Reference

### Text Field Limits

| Field | Sentences | ~Characters |
|-------|-----------|-------------|
| Life/Motivations | 2-3 | 150-250 |
| Personality/Values | 2-3 | 150-250 |
| Layer 2 fields | 2-3 each | 150-250 each |
| Current Solutions | 2-3 | 200-300 |

### List Limits

| Field | Bullets |
|-------|---------|
| Tasks/Activities | 4-6 |
| Pains/Lacks | 4-6 |
| Expectations/Hopes | 4-6 |

### Special Constraints

| Element | Limit |
|---------|-------|
| Need Statement | 60 characters max |
| Age | Exact number (not range) |
| Name | Specific first name |

## Need Statement Examples

**Good Examples:**
- "Reduce data collection redundancy" [32 chars]
- "Standardize validation framework" [31 chars]
- "Automate progress tracking" [25 chars]
- "Achieve 90% partner satisfaction" [32 chars]

**Problematic Examples:**
- "Significantly improve the overall process efficiency" [52 chars] - Too vague
- "Want better system" [17 chars] - Too generic
- "Need our product to help them" [29 chars] - Solution-focused

---

## Quick Entry Workflow

1. **Copy Layer 1** - Name, Age, Life/Motivations, Personality/Values
2. **Copy Layer 2** - All four quadrants
3. **Copy Layer 3** - Tasks, Pains, Hopes
4. **Enter Need Statement** - Verify <60 chars
5. **Copy Layer 4** - Current Solutions
6. **Verify formatting** - No markdown, proper limits
7. **Check badge** - Update validation status in system
