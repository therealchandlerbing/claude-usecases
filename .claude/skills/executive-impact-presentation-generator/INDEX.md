## What This Skill Does

Generates **two versions** of your executive impact report from a single content input:

**Presentation Format** (Landscape Slides)
- Fixed-slide deck for board meetings
- One page per topic
- Optimized for live presentation
- Exports as landscape PDF

**Executive Format** (Portrait Pages)
- Continuous-scroll document for sharing
- Unlimited content per section
- Optimized for comprehensive review
- Exports as portrait PDF

**Same content. Two optimized formats. One request.**

## The Six Sections

Every impact report includes:

1. **Impact Overview**
   - Lives impacted, countries, capital
   - Year-over-year momentum

2. **Regional Portfolio**
   - Geographic allocation (6 regions)
   - Strategic priorities

3. **Financial Performance**
   - Capital structure and efficiency
   - Mix and safeguards

4. **Program Outcomes**
   - Three impact pillars
   - Evaluation approach

5. **Partnership Ecosystem**
   - Collaboration statistics
   - Flagship partnerships

6. **Strategic Outlook**
   - Three-horizon roadmap
   - Leadership asks

## Presentation vs Executive

| Feature | Presentation | Executive |
|---------|-------------|-----------|
| **Layout** | Fixed slides | Continuous scroll |
| **Height** | ~700px limit | Unlimited |
| **Best For** | Live meetings | Email sharing |
| **Print** | Landscape deck | Portrait report |
| **Use When** | Board presentation | Comprehensive review |

**Pro tip**: Generate both. Use presentation for meetings, executive for distribution.

## How to Export PDFs

**Presentation (Landscape Deck):**
1. Open HTML in Chrome
2. Cmd+P (Mac) or Ctrl+P (Windows)
3. Select "Save as PDF"
4. Set to **Landscape**
5. Enable "Background graphics"
6. Save

**Executive (Portrait Report):**
1. Open HTML in Chrome
2. Cmd+P (Mac) or Ctrl+P (Windows)
3. Select "Save as PDF"
4. Set to **Portrait**
5. Enable "Background graphics"
6. Save
