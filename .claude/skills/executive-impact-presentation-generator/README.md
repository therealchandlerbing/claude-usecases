- Landscape slides optimized for presentations

### Executive Format
**Continuous document for comprehensive review**

- Portrait layout for detailed reading
- Sections flow naturally with unlimited height
- Perfect for email distribution, stakeholder sharing
- Section-based navigation with smooth scrolling
- Prints as portrait PDF document (multi-page)

**Same content. Optimized for different contexts. Generated simultaneously.**

## How It Works

### 1. Provide Your Content

Share your organizational impact data in any format:
- Structured (JSON, YAML, spreadsheet)
- Semi-structured (bullet points, descriptions)
- Natural language (conversational)
- Previous report update (delta changes)

**Minimum required:**
- Organization name and brand color (hex code)
- Metrics and descriptions for all six sections

### 2. Skill Generates Both Formats

The skill:
1. Parses your content into standard schema
2. Applies your branding (color, organization name)
3. Populates both template formats
4. Validates completeness and quality
5. Generates two ready-to-use HTML files

### 3. Download and Distribute

Receive two files:
- `[Organization]-Impact-Report-Presentation-[Year].html`
- `[Organization]-Impact-Report-Executive-[Year].html`

**Open in Chrome and export to PDF:**
- Presentation: Landscape orientation (one slide per page)
- Executive: Portrait orientation (multi-page document)

**Distribute:**
- Email executive format to stakeholders
- Present using presentation format in board meetings
- Archive both PDFs for organizational records

**Time from content to delivery: 10-20 minutes**

## Frequently Asked Questions

### Can I customize beyond the six sections?

Yes! While the standard framework includes six sections, you can request:
- Additional subsections within sections
- Custom section ordering
- Modified content groupings
- Emphasis on specific sections

### What if my content doesn't fit in presentation slides?

Presentation slides have height limits (~700px). If content overflows:
- Tighten descriptions and use bullet points
- Move detailed content to executive format only
- Request slide splitting (advanced)
- Focus presentation on highlights, executive on details

### Can I connect to live data sources?

Current version requires manual content input. Future versions will support:
- Google Sheets integration
- Airtable connections
- API integrations
- Real-time dashboard mode

### How do I update quarterly reports?

Maintain your content schema file with current data. Update metrics as new data arrives. Regenerate both formats in minutes with updated content.

### What's the difference from PowerPoint?

This skill generates HTML-based reports optimized for web and PDF, with:
- Dual-format generation (not available in PowerPoint)
- Perfect accessibility (challenging in PowerPoint)
- Self-contained web hosting (PowerPoint requires conversion)
- Faster generation (no manual slide design)
- Responsive mobile design (PowerPoint exports fixed)

### Can I embed these on my website?

Yes! Both HTML files are self-contained and can be hosted on any web server. They work fully offline after initial load.

### What about multi-language support?

Current version is English-only. Future versions will support translation and multi-language generation.
