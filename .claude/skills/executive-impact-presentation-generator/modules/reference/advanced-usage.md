# Advanced Usage

## Customization Beyond Defaults

While the skill provides two standard formats, you can request:

### Content Variations
- Emphasis on specific sections
- Custom section ordering
- Additional subsections within standard framework
- Modified metric groupings
- Alternative badge text

### Design Variations
- Alternative color schemes (secondary colors)
- Modified typography (font sizes, weights)
- Custom spacing and padding
- Organization logo integration
- Modified card styles

### Format Adaptations
- Presentation with more/fewer slides
- Executive with table of contents
- Hybrid format (elements of both)
- Custom print layouts
- Modified navigation styles

## Integration with Other Systems

### Data Source Integration
- Google Sheets (copy/paste or manual pull)
- Airtable (export to CSV, process)
- Excel spreadsheets (structured format)
- CRM systems (export impact metrics)
- Financial systems (capital data)

### Output Integration
- Website embedding (host HTML files)
- Document management systems (PDF archival)
- Email marketing platforms (executive format)
- Board portals (secure distribution)
- Annual report compilation (combine with other materials)

### Workflow Integration
- Quarterly reporting cycles
- Board meeting preparation workflows
- Stakeholder communication calendars
- Grant reporting schedules
- Annual planning processes
