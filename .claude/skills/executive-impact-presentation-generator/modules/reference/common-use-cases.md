# Common Use Cases

## Use Case 1: Quarterly Board Meeting

**Scenario:** Preparing for quarterly board meeting with impact update

**Workflow:**
1. Gather Q3 impact data from program teams
2. Update metrics from previous quarter's report
3. Generate both formats with updated data
4. Email executive format to board 5 days before meeting
5. Present using presentation format during meeting
6. Archive both PDFs for organizational records

**Time Investment:** 30-45 minutes (mostly data gathering)

---

## Use Case 2: Annual Impact Report

**Scenario:** Creating comprehensive annual impact report for stakeholders

**Workflow:**
1. Compile full-year data across all six sections
2. Write momentum highlights showcasing major achievements
3. Generate both formats with full annual data
4. Export both to PDF
5. Email executive format to full stakeholder list
6. Post presentation format on website
7. Print presentation PDF for board book

**Time Investment:** 1-2 hours (comprehensive data compilation)

---

## Use Case 3: Partnership Pitch

**Scenario:** Pitching potential new foundation partner

**Workflow:**
1. Start with existing impact report content
2. Customize regional portfolio to emphasize partner's geography
3. Adjust outcomes section to highlight aligned impact areas
4. Generate presentation format for in-person pitch
5. Generate executive format as leave-behind
6. Tailor strategic outlook to mutual opportunities

**Time Investment:** 45-60 minutes (customization)

---

## Use Case 4: Grant Reporting

**Scenario:** Submitting impact report to major funder

**Workflow:**
1. Adapt content schema to funder's reporting requirements
2. Emphasize programs funded by this specific funder
3. Generate executive format for comprehensive documentation
4. Export to PDF for formal submission
5. Include supplementary data in appendices (manual)

**Time Investment:** 1-2 hours (adaptation to funder requirements)

---

## Use Case 5: Ongoing Updates

**Scenario:** Monthly/quarterly updates with changing metrics

**Workflow:**
1. Maintain content schema file with current data
2. Update metrics monthly as new data arrives
3. Regenerate reports as needed (minutes)
4. Distribute updated executive format via email
5. Present updated presentation format in meetings

**Time Investment:** 10-15 minutes per update (metric refresh only)
