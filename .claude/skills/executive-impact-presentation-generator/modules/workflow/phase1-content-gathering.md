# Phase 1: Content Gathering (5-8 minutes)

## Step 1: Initial Context

Ask the user to provide:
1. Organization name
2. Brand color (hex code)
3. Fiscal year or reporting period
4. Report audience (Board, Funders, Stakeholders)
5. Any specific emphasis areas

## Step 2: Section-by-Section Content Collection

For each of the six sections, gather required content:

### Section 1: Impact Overview
- 3 reach metrics (label, value, description)
- 3-4 momentum highlights (title, badge, description)

### Section 2: Regional Portfolio
- 6 regions (name, percentage, description)
- 3 strategic priorities (title, description)

### Section 3: Financial Performance
- 4-6 capital metrics (metric, value, commentary)
- Capital mix breakdown (category, percentage, description)
- 2-3 safeguards (title, description)

### Section 4: Program Outcomes
- 3 impact pillars (name, metric label, metric value, 3-4 outcomes each)
- Evaluation approach (3 principles with descriptions)

### Section 5: Partnership Ecosystem
- Partnership statistics (3 overall stats)
- Partner categories (4-6 categories with counts)
- Flagship partnerships (4-6 with status badges)

### Section 6: Strategic Outlook
- 3-horizon roadmap (6-18-36 month view with descriptions)
- Leadership asks (3-4 priorities with descriptions)

## Step 3: Content Validation

- Check completeness of all sections
- Verify metrics formatted correctly
- Ensure descriptions are clear
- Validate length for presentation format
- Confirm brand color is valid hex code

## Input Flexibility

The skill accepts content in multiple formats:
- **Structured data** (JSON, YAML, spreadsheet)
- **Semi-structured text** (bullet points, descriptions)
- **Natural language description** (conversational)
- **Previous report update** (delta changes)
- **Mixed formats** (combine approaches)
