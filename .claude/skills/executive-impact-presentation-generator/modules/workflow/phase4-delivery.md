# Phase 4: Delivery (1-2 minutes)

## Step 1: Generate Files

- Create presentation HTML file
- Create executive HTML file
- Name with convention: `[Organization]-Impact-Report-[Format]-[Year].html`

### File Naming Examples
- `Acme-Foundation-Impact-Report-Presentation-2025.html`
- `360-Studios-Impact-Report-Executive-FY2024.pdf`

## Step 2: Provide User Instructions

### Explain Both Formats

**Presentation Format:**
- Fixed-slide layout for board meetings
- Landscape orientation
- One slide per page when printed
- PowerPoint alternative for live display

**Executive Format:**
- Continuous-scroll document for detailed review
- Portrait orientation
- Multi-page document when printed
- Email-friendly format for distribution

### PDF Export Instructions

**Presentation Format (Landscape Deck):**
1. Open HTML file in Chrome
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows)
3. **Destination**: Save as PDF
4. **Layout**: Landscape
5. **Pages**: All
6. **Margins**: None
7. **Options**: Enable "Background graphics"
8. **Scale**: 100% (default)
9. Click "Save"

**Result:** 7-page landscape PDF (cover + 6 section slides)

**Executive Format (Portrait Document):**
1. Open HTML file in Chrome
2. Press `Cmd+P` (Mac) or `Ctrl+P` (Windows)
3. **Destination**: Save as PDF
4. **Layout**: Portrait
5. **Pages**: All
6. **Margins**: Default or None
7. **Options**: Enable "Background graphics"
8. **Scale**: 100% (default)
9. Click "Save"

**Result:** Multi-page portrait PDF (varies by content length)

## Step 3: Offer Follow-up

- Answer questions about specific sections
- Provide customization options
- Offer content refinement
- Suggest distribution strategies
- Note update procedures for future reports

## Distribution Strategies

### Board Meetings
1. Generate both formats
2. Email executive format 3-5 days before meeting (advance reading)
3. Present using presentation format during meeting (live display)
4. Share presentation PDF afterward for archival

### Stakeholder Communications
1. Generate executive format (comprehensive)
2. Email as HTML or PDF attachment
3. Optionally host on website for public access
4. Include in annual report package

### Funder Reporting
1. Generate executive format
2. Customize content for funder requirements
3. Export to PDF for formal submission
4. Maintain HTML version for web-based portals

### Quarterly Updates
1. Update metrics in content schema
2. Regenerate both formats quickly
3. Distribute via email (executive format)
4. Present to board (presentation format)
