# Format Comparison

## Side-by-Side Comparison

| Feature | Presentation Format | Executive Format |
|---------|-------------------|------------------|
| **Layout** | Fixed slides (1050px Ã— 700px) | Continuous scroll sections |
| **Orientation** | Landscape | Portrait |
| **Navigation** | Arrows, dots, keyboard (slide-to-slide) | Section dropdown, smooth scroll |
| **Content Height** | Must fit within slide boundaries | Unlimited, expands as needed |
| **Best For** | Board meetings, live presentations | Email distribution, detailed review |
| **Print Output** | One slide per page (landscape) | Multi-page document (portrait) |
| **Use Case** | PowerPoint alternative for meetings | Comprehensive document sharing |
| **File Size** | ~180KB | ~190KB |
| **Load Time** | < 1 second | < 1 second |
| **Offline** | Fully functional | Fully functional |

## Selection Guidance

### Choose Presentation Format When:
- Live board meeting presentation needed
- One-slide-per-topic layout preferred
- PDF deck format required
- Linear slide progression expected
- Meeting facilitator will advance slides
- Time-limited presentation (constrained content)

### Choose Executive Format When:
- Email distribution to stakeholders
- Comprehensive reading expected
- Sections can be lengthy (no height limit)
- Readers may skip around sections
- Detailed information must be preserved
- Document archival needed

### Generate Both When:
- Different audiences (meeting + email)
- Flexibility in distribution needed
- Complete documentation package required
- Annual archival with multiple uses
- Some content too detailed for slides

## Distribution Recommendations

**For Quarterly Board Meetings:**
- Email executive format 3-5 days before meeting (advance reading)
- Present using presentation format during meeting (live display)
- Share presentation PDF afterward for archival

**For Annual Stakeholder Reports:**
- Generate both formats
- Email executive format to full stakeholder list
- Post presentation format on website
- Print presentation PDF for board book

**For Partnership Pitches:**
- Generate presentation format for in-person pitch
- Generate executive format as leave-behind
- Tailor strategic outlook to mutual opportunities

**For Funder Reporting:**
- Generate executive format for comprehensive documentation
- Export to PDF for formal submission
- Maintain HTML version for web-based portals
