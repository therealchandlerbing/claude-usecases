# Dual-Format Output System

## Format 1: Presentation (Slide Deck)

**Characteristics:**
- Fixed-slide layout (1050px Ã— 700px per slide)
- Landscape orientation for meetings and board presentations
- Slide-based navigation (arrows, dots, keyboard, dropdown)
- Content constrained to fit within slide boundaries
- One slide per page when printed
- PowerPoint alternative for live display

**Navigation Methods:**
- Arrow keys (left/right) - Previous/next slide
- Navigation dots - Jump to specific slide
- Keyboard shortcuts - Home (first), End (last), Space (next)
- Slide dropdown menu - Select any slide by name

**Best For:**
- Live board meeting presentation
- One-slide-per-topic layout
- PDF deck format
- Linear slide progression
- Meeting facilitator will advance slides
- Time-limited presentation

## Format 2: Executive (Continuous Document)

**Characteristics:**
- Continuous-scroll sections with unlimited height
- Portrait orientation for comprehensive review
- Section-based navigation with smooth scrolling
- Content can expand to any length per section
- Multi-page document when printed
- Email-friendly format for distribution

**Navigation Methods:**
- Section dropdown menu - Jump to any section
- Smooth scrolling - Natural reading flow
- Keyboard scrolling - Page up/down, arrow keys

**Best For:**
- Email distribution to stakeholders
- Comprehensive reading
- Sections can be lengthy (no height limit)
- Readers may skip around sections
- Detailed information must be preserved
- Document archival

## Shared Foundation

**Both formats share:**
- Same content source feeds both formats
- Consistent branding and visual identity
- Identical navigation paradigms adapted to format
- Self-contained HTML with embedded CSS
- Google Fonts only external dependency
- Works fully offline after initial load

**Technical Specifications:**
- Valid HTML5 structure
- WCAG 2.1 AA accessibility compliant
- Responsive design (desktop, tablet, mobile)
- Browser compatible (Chrome, Firefox, Safari, Edge)
- File size typically < 200KB
- Load time typically < 1 second

## When to Generate Both

Generate both formats when:
- Different audiences (meeting + email)
- Flexibility in distribution needed
- Complete documentation package required
- Annual archival with multiple uses
- Some content too detailed for slides
