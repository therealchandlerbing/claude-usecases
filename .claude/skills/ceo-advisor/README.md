## Why CEO Advisor?

### The Executive Challenge

Modern CEOs face an impossible bandwidth problem:

- **Information Overload**: Hundreds of signals competing for attention
- **Relationship Complexity**: Dozens of critical stakeholders requiring personalized engagement
- **Time Scarcity**: Only 20% of time spent on truly strategic work
- **Decision Fatigue**: 35,000+ decisions daily leading to cognitive depletion
- **Reactive Patterns**: Constantly firefighting instead of anticipating

### The CEO Advisor Solution

An integrated advisory team that works while you sleep:

**Predictive Intelligence**
- Early warning system detects issues 72+ hours before they become crises
- Pattern recognition across organizational silos
- Automated priority scoring for your attention

**Relationship Optimization**
- AI-powered sentiment analysis across all stakeholder interactions
- Personalized engagement strategies by communication style
- Proactive intervention recommendations for at-risk relationships

**Performance Maximization**
- Calendar optimization aligned with your energy patterns
- Meeting efficiency scoring with consolidation recommendations
- Protected focus time for strategic thinking

**Strategic Clarity**
- Scenario modeling for major decisions
- Competitive intelligence synthesis
- Risk-adjusted option analysis

**Financial Oversight**
- Real-time financial health monitoring
- Resource allocation optimization
- Cash flow forecasting with decision triggers

## Primary Use Cases

### Daily Intelligence Brief
Start every morning with a comprehensive brief covering critical issues, stakeholder health, schedule optimization, and prioritized actions.

```
"Generate my morning intelligence brief"
```

### Board Meeting Preparation
Complete preparation including individual board member analysis, predicted questions with preparation strategies, narrative development, and risk identification.

```
"Prepare me for the board meeting next Tuesday"
```

### Crisis Response Protocol
Immediate crisis management with severity assessment, action plan, stakeholder communication strategy, and recovery roadmap.

```
"We have a crisis: [describe situation]"
```

### Strategic Planning Session
Comprehensive strategic analysis with environmental scanning, option generation, financial modeling, and implementation roadmap.

```
"Facilitate a strategic planning session on [topic]"
```

### Stakeholder Relationship Review
Portfolio analysis with health scoring, risk matrix, trajectory predictions, and personalized engagement recommendations.

```
"Review my stakeholder relationships and identify concerns"
```

### Schedule Optimization
Time allocation analysis with meeting efficiency scoring, energy optimization, and delegation recommendations.

```
"Optimize my schedule for next week"
```

## Getting Started

### Prerequisites
- Python 3.8+
- Claude Code with managed skills enabled

### Installation

```bash
# Navigate to skills directory
cd .claude/skills/ceo-advisor

# Install Python dependencies
pip install -r config/requirements.txt

# Generate sample data
python examples/sample_data_generator.py
```

### First Run

```bash
# Generate your first daily intelligence brief
python src/ceo_advisor_orchestrator.py daily
```

### Invoke as Skill

When using with Claude Code, simply invoke the skill naturally:

```
"I need CEO advisory support for my upcoming board meeting"
```
