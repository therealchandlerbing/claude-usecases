# CEO Advisor - Executive Frameworks

Reusable templates and frameworks for executive decision-making.

---

## Decision-Making Frameworks

### Framework 1: RAPID Decision Framework

Use this for decisions requiring clear accountability.

```
DECISION: [State the decision to be made]

| Role | Person | Responsibility |
|------|--------|----------------|
| **R**ecommend | [Name] | Proposes the decision, gathers input |
| **A**gree | [Names] | Must agree (can veto) |
| **P**erform | [Names] | Executes the decision |
| **I**nput | [Names] | Consulted for expertise |
| **D**ecide | [Name] | Final decision maker |

Timeline: [Date needed]

Escalation: If no agreement by [date], escalate to [person]
```

### Framework 2: 10/10/10 Decision Test

Use for emotionally charged or high-stakes decisions.

```
DECISION: [State the decision]

How will I feel about this decision:
- In 10 minutes? [Immediate reaction]
- In 10 months? [Medium-term impact]
- In 10 years? [Long-term significance]

Reversibility Assessment:
- Easily reversible: [ ]
- Partially reversible: [ ]
- Irreversible: [ ]

Proceed: [ ] Yes  [ ] No  [ ] Need more information
```

### Framework 3: Pre-Mortem Analysis

Use before major initiatives to identify risks.

```
INITIATIVE: [Name]

Imagine it's [date + timeline] and this initiative has failed spectacularly.

What went wrong?
1. [Failure mode 1]
2. [Failure mode 2]
3. [Failure mode 3]

Most likely cause of failure:
[Description]

Mitigation actions:
1. [Action to prevent failure mode 1]
2. [Action to prevent failure mode 2]
3. [Action to prevent failure mode 3]

Early warning signals to monitor:
1. [Signal 1]
2. [Signal 2]
```

## How to Use These Frameworks

1. **Copy** the relevant template
2. **Customize** for your specific context
3. **Fill in** with your information
4. **Review** with CEO Advisor for optimization
5. **Iterate** based on results

These frameworks are designed to be:
- **Scannable** - Quick to review
- **Complete** - Cover essential elements
- **Actionable** - Drive specific outcomes
- **Repeatable** - Use consistently over time

Customize terminology and sections to match your organization's language and culture.
