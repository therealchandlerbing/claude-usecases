# CEO Advisor - AI Prompt Templates

Quick reference prompts for interacting with the CEO Advisor system.

## Daily Operations

### Morning Intelligence Brief

```
Generate my daily executive intelligence brief. Include:
- Critical issues requiring immediate attention (red flags)
- Items to monitor (yellow flags)
- Opportunities to capitalize on (green flags)
- Key stakeholder touchpoints needed today
- Optimal schedule for energy and impact
- Top 3 decisions requiring my input
- Predictive alerts for next 48 hours
```

### Board Meeting Preparation

```
Prepare me for the upcoming board meeting on [DATE]. Analyze:
- Each board member's current sentiment and concerns
- Likely questions based on recent patterns
- Optimal narrative flow for maximum impact
- Risk areas requiring proactive addressing
- Relationship dynamics to navigate
- Energy management strategy for the meeting
- Post-meeting follow-up priorities
```

### Crisis Response

```
We're facing [CRISIS DESCRIPTION]. Provide:
- Immediate action plan (0-24 hours)
- Stakeholder communication strategy with sentiment predictions
- Resource allocation recommendations
- Decision framework for rapid response
- Energy preservation tactics during crisis
- Early warning signals to monitor
- 30-60-90 day recovery roadmap
```

## Tips for Effective Prompts

1. **Be Specific**: Include context, constraints, and desired outcome
2. **Set Time Frames**: Specify deadlines and planning horizons
3. **Define Success**: State how you'll measure results
4. **Include Data**: Reference specific metrics or situations
5. **Request Format**: Specify how you want information presented
6. **Iterate**: Refine prompts based on what's most useful
