# 360 Board Meeting Prep - Quick Reference

**One-page guide for rapid onboarding**

---

## Trigger Phrases

```
"Help me prep for the [date] board meeting"
"Create a board packet for [date]"
"Build board meeting materials"
"Generate board documents"
```

---

## What You Get

### Documents (5)
1. Financial Dashboard
2. Client Portfolio Health
3. Strategic Initiatives
4. Governance & Compliance
5. Motion Tracking Sheet

### Formats (3)
- **DOCX** - Master board packet
- **HTML** - Interactive dashboards
- **PDF** - Print-ready exports

## Timeline

| Day | Phase |
|-----|-------|
| T-10 | Data collection, gap identification |
| T-7 | Draft generation, cross-validation |
| T-7 to T-2 | User review, refinement |
| T-2 | Final generation, design elevation |
| T+2 | Post-meeting follow-up |

## Data Sources

| Source | What's Pulled |
|--------|--------------|
| **Asana** | Pipeline projects, action items |
| **QuickBooks** | Revenue, expenses, AR, cash |
| **Gmail** | Partner communication patterns |
| **Drive** | Historical board packets |

---

## Required Asana Fields

For pipeline projects:
- `contract_value`
- `probability`
- `stage`
- `client_tier`

---

## Design-Director Integration

**Automatic elevation on all outputs**:
- Stripe exemplar for financial content
- Swiss design for layouts
- Typography hierarchy
- Print optimization

---

## Flag System

| Flag | Meaning |
|------|---------|
| Cash <6 months | Runway warning |
| AR >60 days | Collection needed |
| Variance >20% | Budget attention |
| No updates >14 days | Stale data |
