# Workflow Modules

This directory contains the 6-phase workflow for board meeting preparation:

## Phase Sequence

1. **phase1-data-collection.md** (T-10 days) - Initial consultation, data gathering, validation
2. **phase2-document-generation.md** (T-7 days) - Generate 5 draft documents
3. **phase3-review-refinement.md** (T-7 to T-2 days) - User feedback and iterations
4. **phase4-final-output.md** (T-2 days) - DOCX, HTML, PDF generation with design elevation
5. **phase5-meeting-support.md** (Meeting Day) - Real-time meeting assistance
6. **phase6-post-meeting-followup.md** (T+2 days) - Action items, archival, minutes

## Typical Timeline

- **10 days before meeting**: Start Phase 1
- **7 days before**: Complete Phase 2 drafts
- **7-2 days before**: Phase 3 refinement cycles
- **2 days before**: Phase 4 final outputs
- **Meeting day**: Phase 5 support
- **2 days after**: Phase 6 follow-up

Load phases sequentially as workflow progresses.
