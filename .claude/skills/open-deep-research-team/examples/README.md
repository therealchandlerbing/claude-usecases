## Common Patterns

### What Makes Research Successful
1. Clear objectives and decision context
2. Multi-perspective analysis (academic + technical + data)
3. Explicit contradiction analysis
4. Actionable recommendations
5. Appropriate depth for stakes

### Typical Research Workflow
1. Query clarification (if needed)
2. Research brief generation
3. Parallel specialist deployment
4. Finding synthesis
5. Report generation
6. Recommendations and next steps
