Usage Notes

How to use these components:

  1. Copy the HTML and CSS for any component you want to use
  2. Paste into your portfolio page
  3. Customize content (text, numbers, colors)
  4. Adjust spacing and sizing as needed
  5. Test on mobile devices (max-width: 768px)

Best Practices:

  • Don't use all components on one page—select what fits your story
  • Vary layouts throughout the page (don't repeat same pattern)
  • Ensure color variables match your base-template.html
  • Test accessibility (color contrast, keyboard navigation)
  • Keep mobile users in mind (test on small screens)
