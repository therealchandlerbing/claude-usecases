## Quick Reference Cards

### For Data Collection

**Must-Have Information:**
1. Team name and tagline
2. Problem statement (2-3 sentences)
3. Solution approach (2-3 sentences)
4. Current stage (TRL + validation)
5. 3-5 key milestones
6. Target audience
7. 360 partnership context

**Reference:** [vianeo-intake-form.md](templates/vianeo-intake-form.md)

### For Design Direction

**360 Brand Colors:**
- Sage: #87A878
- Terracotta: #C97A63
- Plum: #6B5B95
- Gray: #4A5859

**Client Colors by Sector:**
- CleanTech: Green (#047857) + Blue (#0EA5E9)
- HealthTech: Blue (#3B82F6) + Coral (#FB7185)
- EdTech: Purple (#9333EA) + Yellow (#F59E0B)

**Typography:**
- Serif: Crimson Pro / Lora / Fraunces
- Sans: Inter / Work Sans / DM Sans

**Reference:** [design-standards.md](references/design-standards.md)

### For Content Translation

**TRL Translations:**
- TRL 1-2: "Exploring concept"
- TRL 3-4: "Building prototype"
- TRL 5-6: "Pilot testing"
- TRL 7-8: "Scaling proven model"
- TRL 9: "Mature technology"

**Reference:** [vianeo-translation-guide.md](references/vianeo-translation-guide.md)

### For Quality Check

**Before Launch:**
- [ ] Value prop clear in 10 seconds
- [ ] Loads under 3 seconds on 4G
- [ ] Mobile responsive (375px to 1920px)
- [ ] WCAG AA accessible
- [ ] No console errors
- [ ] Content understandable without technical background

**Reference:** [quality-checklist.md](references/quality-checklist.md)

**New to skill:**

### "Portfolio looks too corporate"

**Solutions:**
1. Review [design-standards.md](references/design-standards.md) - Asymmetry principles
2. Study [examples/](examples/) - Hand-crafted feel
3. Reference [IMPLEMENTATION-GUIDE.md](IMPLEMENTATION-GUIDE.md) - Editorial sophistication

**Solutions:**

2. Apply [content-strategy.md](references/content-strategy.md) - Accessibility principles
3. Check [sector-positioning.md](references/sector-positioning.md) - Industry framing

**Solutions:**
