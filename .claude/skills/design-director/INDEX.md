## Quick Reference

### Supported Output Types

1. **Presentations** - PowerPoint, Google Slides, Keynote
2. **Spreadsheets** - Excel, Google Sheets
3. **HTML/Web** - Static sites, dashboards, web apps
4. **Reports** - Word, Google Docs, Markdown
5. **Dashboards** - Data visualization, analytics

### Key Exemplars

| Exemplar | Best For | Key Color |
|----------|----------|-----------|
| **Stripe** | Financial, dashboards, data | #0A2540 |
| **Linear** | Product tools, SaaS | #5E6AD2 |
| **Apple** | Premium, elegant, minimal | #000000 |
| **Notion** | Content, collaborative | Warm whites |
| **Bauhaus** | Bold, geometric | Primary RGB |
| **Swiss** | Systematic, typography | #FF0000 accent |

## Skill Workflow

```
User provides output + context
        ↓
Design Interrogation (evaluate all choices)
        ↓
Select exemplar (Stripe/Linear/Apple/etc.)
        ↓
Choose 2-3 techniques
        ↓
Apply systematically throughout
        ↓
Validate against quality standards
        ↓
Deliver polished result
```
