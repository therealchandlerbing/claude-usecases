# 360 Brand Quality Checklist

**Pre-Delivery Validation for All 360 Materials**

Use this checklist before finalizing any document, presentation, report, or visual output for 360 Social Impact Studios.

---

## Logo & Visual Identity

### Logo Presence
- [ ] Logo is present on document
- [ ] Correct logo variation used for document type (see Quick Reference)
- [ ] Logo properly sized (meets minimum requirements)
  - Full logo: minimum 2 inches / 150px
  - Circle badge: minimum 0.75 inches / 75px
- [ ] Logo placed in correct location
  - Reports: Top left
  - Presentations (title): Top center
  - Presentations (content): Top right (circle badge)
  - One-pagers: Top center

### Logo Quality
- [ ] Logo has adequate clear space (equal to height of "360")
- [ ] Logo is not stretched, distorted, or modified
- [ ] Logo colors unchanged (no filters, tints, or effects)
- [ ] Logo resolution appropriate (150 DPI minimum)
- [ ] If dark background, dark logo version used

### Visual Brand Elements
- [ ] Graphic mark (if used) at appropriate opacity (10-30%)
- [ ] Brand patterns used intentionally, not decoratively
- [ ] Visual elements support content, don't overwhelm

---

## Color Palette

### Color Accuracy
- [ ] All colors use exact 360 hex codes (not approximations)
  - Navy Blue: #1E4D6B
  - Bright Blue: #00A7E1
  - Lime Green: #7AC143
  - Magenta: #E6007E
  - Red: #ED1C24
  - Orange: #FF8C00
  - Yellow: #FFD100 (backgrounds only!)
  - Black: #231F20
  - Light Gray: #F5F5F5

### Color Application
- [ ] Headers in Navy Blue (#1E4D6B) or Bright Blue (#00A7E1)
- [ ] Body text in Black (#231F20)
- [ ] Accent colors match audience needs
- [ ] No more than 2-3 accent colors per section
- [ ] **Critical:** No yellow text on white backgrounds
- [ ] Data visualizations use 360 color sequences

### Audience-Appropriate Palette
- [ ] **Board materials:** Navy + Bright Blue + White only
- [ ] **Partnership (corporate):** Navy + Bright Blue professional
- [ ] **Partnership (Brazil):** Orange + Magenta + Yellow warm
- [ ] **Community:** Vibrant accents appropriate
- [ ] **Internal:** Functional colors appropriate

---

## Typography

### Font Family
- [ ] Headers use Montserrat Bold or Arial Bold (fallback)
- [ ] Body text uses Open Sans Regular or Arial Regular (fallback)
- [ ] Consistent font family throughout document
- [ ] No more than 2 font families total

### Typography Hierarchy
- [ ] Level 1 titles: 28-36pt, Bold, Navy Blue or Black
- [ ] Level 2 headers: 18-22pt, Bold/SemiBold, Navy Blue or Bright Blue
- [ ] Level 3 subheaders: 14-16pt, SemiBold, Navy Blue or Black
- [ ] Body text: 10-12pt, Regular, Black
- [ ] Captions: 8-9pt minimum, Regular, Gray

### Typography Quality
- [ ] Appropriate line height (1.5-1.6 for body text)
- [ ] Proper heading hierarchy (H1 > H2 > H3)
- [ ] No orphans or widows (single lines at top/bottom of pages)
- [ ] Text alignment intentional (not centered arbitrarily)
- [ ] Headers colored appropriately (Navy or Bright Blue, not black)

---

## Content & Structure

### Document Structure
- [ ] Clear title/headline
- [ ] Logical section organization
- [ ] Proper heading hierarchy throughout
- [ ] Adequate white space and margins
- [ ] No awkward page breaks or section breaks

### Required Elements (by type)
**Reports:**
- [ ] Title page with full information
- [ ] Page numbers
- [ ] Executive summary (if multi-page)
- [ ] Clear section breaks

**Presentations:**
- [ ] Complete title slide
- [ ] Consistent slide layout
- [ ] Section dividers (if multi-section)
- [ ] Not overcrowded (max 3-5 bullets per slide)

**One-Pagers:**
- [ ] Most important info prominent
- [ ] Clear visual hierarchy
- [ ] Call to action included
- [ ] Single page (no continuation)

### Footer/Header
- [ ] Page numbers (if multi-page)
- [ ] Date and version (if applicable)
- [ ] Contact info (if appropriate)
- [ ] Optional circle badge in footer

## Accessibility (WCAG AA)

### Color Contrast
- [ ] Text meets 4.5:1 contrast minimum
- [ ] Navy on white: PASS ✓
- [ ] Magenta on white: PASS ✓
- [ ] Black on white: PASS ✓
- [ ] Bright Blue on white: **Only large text (18pt+ or 14pt bold+) or dark backgrounds**
- [ ] Lime Green on white: **Only large text (18pt+ or 14pt bold+) or dark backgrounds**
- [ ] **Yellow on white: FAIL** ❌ (not used for text)

### Font Sizes
- [ ] Body text minimum 10pt (12pt preferred)
- [ ] Caption text minimum 8pt
- [ ] Presentation text minimum 16pt
- [ ] High-importance text 14pt+

### Accessibility Features
- [ ] Alt text provided for all images
- [ ] Proper heading hierarchy (not skipping levels)
- [ ] PDFs screen-reader accessible (if PDF)
- [ ] Information not conveyed by color alone
- [ ] Links descriptive (not "click here")
- [ ] Focus states present (web/interactive)
- [ ] Touch targets 44x44px minimum (web)

## Audience Alignment

### Tone
- [ ] **Partners/Funders:** Professional, strategic, data-driven
- [ ] **Board:** Conservative, transparent, metrics-focused
- [ ] **Community:** Accessible, warm, inclusive, empowering
- [ ] **Internal:** Collaborative, action-oriented, supportive

### Language
- [ ] Appropriate for audience sophistication
- [ ] Technical vs. accessible language match
- [ ] Jargon avoided (or explained if necessary)
- [ ] Inclusive language used
- [ ] Systems-thinking lens present (where appropriate)

### Cultural Considerations
- [ ] Regional preferences considered
  - Brazil: Warm, vibrant, relationship-focused
  - Corporate: Professional, data-driven
  - Sustainability: Environmental themes
- [ ] Diverse representation in imagery
- [ ] Culturally sensitive content

## Document Type Specifications

### Reports/Documents
- [ ] Margins: 1 inch all sides
- [ ] Page size: 8.5 x 11 inches (US Letter)
- [ ] Orientation: Portrait (or landscape if justified)
- [ ] Logo in header: Top left, 2 inches
- [ ] Proper footer with page numbers

### Presentations
- [ ] Format: 16:9 widescreen
- [ ] Title slide logo: Center, 3-4 inches
- [ ] Content slide logo: Top right, 1-1.5 inches (circle badge)
- [ ] Text readable from distance (16pt minimum)
- [ ] Consistent slide master applied

### One-Pagers
- [ ] Logo: Top center, 2.5-3 inches
- [ ] 2-3 column layout (or justified alternative)
- [ ] Key callouts prominent (16-18pt with accent colors)
- [ ] Footer with contact info + small circle badge
- [ ] Fits on single page

## Design Quality

### Hand-Crafted Appearance
- [ ] Deliberate white space (not default padding)
- [ ] Typography relationships (not just bigger/bold)
- [ ] Color relationships intentional (not random)
- [ ] Visual hierarchy clear (3 levels minimum)
- [ ] Considered alignment (not just left-everything)
- [ ] Doesn't look template-based

### Design Director Standards Met
- [ ] Would pass as portfolio-ready quality
- [ ] No red flags:
  - [ ] Not using defaults without consideration
  - [ ] Not more than 4 colors in palette
  - [ ] Clear visual hierarchy present
  - [ ] Typography varies in size/weight
  - [ ] White space not cramped
  - [ ] Couldn't swap with template unnoticed

### Consistency
- [ ] Consistent spacing throughout
- [ ] Consistent color usage
- [ ] Consistent typography application
- [ ] Consistent header/footer placement
- [ ] All pages/slides follow same structure

## Data Visualization (if applicable)

### Chart/Graph Quality
- [ ] Uses 360 color sequences
  - Single: Bright Blue
  - 2-3: Navy, Bright Blue, Lime Green
  - 4+: Navy, Bright Blue, Lime Green, Magenta, Orange, Yellow
- [ ] Clean, minimal gridlines (light gray)
- [ ] Clear axis labels (black or dark gray, legible size)
- [ ] Legend positioned logically (top right or bottom)
- [ ] Data labels included where space allows
- [ ] No 3D effects or shadows

### Table Quality
- [ ] Clear header distinction (bold + background)
- [ ] Numbers right-aligned for comparison
- [ ] Zebra striping (subtle) OR minimal borders
- [ ] Readable font size
- [ ] Proper spacing (not cramped)

---

## Technical Quality

### File Quality
- [ ] Saved in appropriate format
  - Print: PDF (300 DPI)
  - Digital: PDF (150 DPI) or native format
  - Collaboration: DOCX, PPTX, etc.
- [ ] Proper file resolution for use case
- [ ] File named according to convention
- [ ] Version number included (if versioned)

### Functional Elements
- [ ] All links tested and working (if digital)
- [ ] Images high resolution (no pixelation)
- [ ] No placeholder text remaining ("Lorem ipsum", "[Insert]", etc.)
- [ ] All data accurate and verified
- [ ] Formulas/calculations correct (if spreadsheets/tables)

### Final Review
- [ ] Spell-check completed
- [ ] Grammar reviewed
- [ ] Names and titles correct
- [ ] Numbers and data verified
- [ ] Consistent style throughout
- [ ] Second person review (if possible)

## Common Issues & Fixes

| Issue | Quick Fix |
|-------|-----------|
| Logo too small | Increase to minimum: 2" (full), 0.75" (badge) |
| Logo distorted | Restore aspect ratio, don't stretch |
| Colors off-brand | Use exact hex codes from Quick Reference |
| Yellow text on white | Change to background use only, or switch color |
| Text too small | Increase: 10pt min body, 8pt min captions, 16pt min presentations |
| Poor contrast | Use approved combinations: Navy/Bright Blue/Magenta/Lime on white |
| Too many colors | Limit to 2-3 accent colors per section |
| Wrong audience palette | Board: conservative. Community: vibrant. Brazil: warm. |
| No clear hierarchy | Apply L1 (28-36pt), L2 (18-22pt), L3 (14-16pt) |
| Template-like | Add deliberate spacing, intentional color use, custom treatments |
