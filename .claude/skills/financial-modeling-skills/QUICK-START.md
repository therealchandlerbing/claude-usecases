## Excel Model Standards

Every financial model includes:

**Core Worksheets:**
1. **Executive Summary** - Deal overview, key metrics, recommendation
2. **Assumptions** - All inputs (blue text, light blue background)
3. **Financial Model** - P&L, cash flow, balance sheet projections
4. **Scenarios** - Base, upside, downside cases
5. **Sensitivity** - Key variable testing
6. **Valuation** - Entry, exit, interim valuations
7. **Returns** - IRR, MOIC, DPI by scenario
8. **Comps** - Trading and transaction comparables
9. **Risks** - Categorized with mitigations

**Formatting:**
- Input cells: Blue text + light blue background
- Calculations: Black text + white background
- References: Green text for cross-sheet links
- Headers: Bold + dark blue + larger font
- Negative values: Red text with parentheses
