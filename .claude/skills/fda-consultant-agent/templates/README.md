## How to Use These Templates

1. **Select Appropriate Template**: Choose based on your submission type
2. **Review Requirements**: Read through all sections to understand scope
3. **Gather Information**: Collect all required data and documentation
4. **Complete Checklist**: Mark items as complete, note gaps
5. **Consult FDA Agent**: Ask specific questions about unclear requirements
6. **Validate Completeness**: Cross-check against FDA guidance before submission

## Template Version Control

All templates are updated quarterly to reflect:
- New FDA guidance documents
- Updated MDUFA/PDUFA performance goals
- Recent FDA policy changes
- Common deficiency patterns
