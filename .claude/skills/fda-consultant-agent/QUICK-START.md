# FDA Consultant Agent - Quick Start Guide

**Goal:** Get expert FDA regulatory guidance in under 5 minutes

**Audience:** First-time users who need immediate FDA consultation

**Time to First Consultation:** 2-5 minutes

### Step 3: Ask Your Question (30 seconds)

**Simple Format:**
```
Using the FDA Consultant Agent, [your question].

Context:
- [Key detail 1]
- [Key detail 2]
- [Key detail 3]
```

**Example:**
```
Using the FDA Consultant Agent, what FDA pathway should I use for a wearable ECG monitor?

Context:
- Continuous heart rhythm monitoring
- Bluetooth connection to smartphone app
- Alerts user and physician of arrhythmias
- Non-invasive, consumer use
```

## üìù Sample Questions to Get Started

### Product Classification

**Medical Device:**
```
What FDA class is a home-use blood glucose monitoring system?
```

**Software:**
```
How would FDA classify AI software that analyzes chest X-rays for pneumonia?
```

**Drug:**
```
What regulatory pathway for a generic version of [brand name drug]?
```

**Combination Product:**
```
Is my drug-eluting stent regulated as a device or drug?
```

### Submission Strategy

**510(k):**
```
What do I need for a 510(k) submission for an insulin pump?
Requirements: Similar to Medtronic MiniMed, but with smartphone control.
```

**PMA:**
```
Timeline and requirements for PMA submission of a left ventricular assist device?
Context: Novel design, Class III, life-sustaining.
```

**IND:**
```
What's required for a Phase 1 IND for a new cancer immunotherapy?
Context: Monoclonal antibody, first-in-human study planned.
```

### Compliance Questions

**Design Controls:**
```
What are the essential elements of a Design History File under 21 CFR 820.30?
Context: Class II medical device, preparing for FDA inspection.
```

**CAPA:**
```
How do I respond to an FDA 483 observation about our CAPA system?
Observation: "CAPA effectiveness checks not adequately documented."
```

**MDR:**
```
Do I need to file an MDR report for this adverse event?
Event: Device malfunction with no patient injury, returned device analysis shows manufacturing defect.
```

### Strategic Planning

**Pathway Selection:**
```
Should I pursue 510(k) or De Novo for a novel digital therapeutic?
Product: Software-based treatment for depression, no direct hardware, prescription-only.
```

**FDA Meeting:**
```
Should I request a Pre-Submission meeting for my Class II diagnostic device?
Situation: Some novel features, unsure about testing requirements.
```

**Timeline:**
```
Realistic timeline from prototype to market for a Class II surgical instrument?
Context: Well-established device category, predicates available.
```

## üéØ Understanding Your Response

### Regulatory Framework Section

**What you'll see:**
```
Classification: Class II Medical Device
Applicable Regulations:
- 21 CFR 870.2340 (Electrocardiograph)
- 21 CFR Part 820 (Quality System Regulation)

Key Guidance:
- "Policy for Device Software Functions" (September 2022)
- "Clinical Decision Support Software" (September 2022)
```

**What this means:** The exact FDA regulations and guidance documents that apply to your product.

### Strategic Recommendations Section

**What you'll see:**
```
Recommended Pathway: 510(k) Premarket Notification

Rationale:
- Class II device with established device type
- Predicates available
- Most time and cost-efficient pathway

Requirements:
- Substantial equivalence demonstration
- Performance testing (bench and clinical)
- Biocompatibility assessment
- Software validation
- Cybersecurity documentation
```

**What this means:** The recommended regulatory pathway, why it's recommended, and what you need to prepare.

### Implementation Steps Section

**What you'll see:**
```
Timeline: 8-12 months
1. Months 1-2: Predicate identification and gap analysis
2. Months 3-5: Testing and validation
3. Months 6-7: Submission preparation
4. Month 8: Submit 510(k)
5. Months 9-12: FDA review
```

**What this means:** Realistic timeline and sequence of activities.

### Critical Considerations Section

**What you'll see:**
```
Risks:
- Cybersecurity concerns due to wireless connectivity
- Potential for clinical data requirement if claims are aggressive

Mitigation:
- Robust cybersecurity documentation per FDA guidance
- Conservative intended use statement
- Consider clinical study even if not required

Success Factors:
- Select recent predicates (within 5 years)
- Comprehensive software documentation
- Early identification of testing gaps
```

**What this means:** Potential problems and how to avoid them.

## üí° Tips for Best Results

### Provide Sufficient Context

**Good:**
```
Wearable continuous glucose monitor, Bluetooth to smartphone, alerts for high/low glucose,
for Type 1 diabetics, similar to Dexcom G7 but with extended wear time (14 days vs 10 days).
```

**Not Enough:**
```
Glucose monitor.
```

### Be Specific About Your Situation

**Good:**
```
We're a startup with a prototype, validated in benchtop testing, planning pilot clinical study.
Need to know FDA pathway before raising Series A funding.
```

**Not Enough:**
```
What do we need to do for FDA?
```

### Ask Follow-Up Questions

After initial guidance, dig deeper:
- "Can you help identify specific predicate devices?"
- "What testing standards apply to this device?"
- "What should be in our Pre-Submission meeting request?"
- "Can you review our proposed intended use statement?"

### Reference Specific Regulations When Needed

If you're already familiar with FDA regulations:
```
Our device seems to fit under 21 CFR 880.5725 (surgical marker).
Can you confirm and explain submission requirements?
```

## üîç Quick Reference: FDA Pathways

### Medical Device Pathways

| Pathway | Device Class | Timeline | When to Use |
|---------|--------------|----------|-------------|
| **Exempt** | Class I (most) | Immediate | Low-risk devices (bandages, gloves) |
| **510(k)** | Class II (most) | 3-6 months | Predicate devices available |
| **De Novo** | Novel Class I/II | 5-7 months | No predicate, low-moderate risk |
| **PMA** | Class III | 12-18 months | High-risk, life-sustaining devices |

### Drug/Biologic Pathways

| Pathway | Type | Timeline | When to Use |
|---------|------|----------|-------------|
| **IND** | Investigational | 30 days | Start clinical trials |
| **NDA** | New Drug | 10-12 months | Novel drug approval |
| **ANDA** | Generic Drug | 10-12 months | Generic version of approved drug |
| **BLA** | Biologic | 10-12 months | Biologics license |
| **505(b)(2)** | Hybrid | 10-12 months | Relies on prior approval data |

## üö® When to Seek Additional Help

### Consult FDA Regulatory Attorney If:
- Warning letter or enforcement action received
- Significant financial or legal liability at stake
- Novel regulatory territory with precedent-setting implications
- Disputed classification or jurisdiction
- Recall or safety issue

### Consult Specialized Consultant If:
- Complex statistical design needed (biostatistician)
- Novel manufacturing process (CMC specialist)
- Specialized testing (biocompatibility lab, EMC testing facility)
- International regulatory strategy (EU Notified Body, PMDA consultant)
- Clinical study design (CRO, clinical specialist)

### Request FDA Meeting If:
- Novel device without clear classification
- Uncertainty about testing requirements
- Substantial disagreement about pathway
- Breakthrough or complex technology
- Want to confirm development plan before major investment

## üìö Next Steps After Quick Start

Once you've received initial guidance:

**1. Deep Dive into Requirements**
- Review referenced guidance documents on FDA.gov
- Read cited CFR sections
- Search FDA databases for predicates

**2. Develop Detailed Plan**
- Create submission timeline with milestones
- Identify testing gaps and vendors
- Budget for user fees, testing, consulting
- Assign internal responsibilities

**3. Build Regulatory Team**
- Identify regulatory lead (internal or consultant)
- Engage testing labs
- Consider legal counsel for strategy
- Plan FDA meeting if needed

**4. Iterate with FDA Consultant Agent**
- Ask follow-up questions as needed
- Request checklist review
- Validate assumptions
- Get guidance on deficiencies

**5. Engage with FDA Directly**
- Submit Q-Submission or Pre-Submission if appropriate
- Attend FDA workshops and webinars
- Join industry associations
- Monitor FDA communications in your device/drug category
