**When you need fast answers:**

## Keeping References Current

### Update Frequency

**Daily:** Monitor for critical changes (safety communications, enforcement)
**Weekly:** New guidance documents, policy updates
**Monthly:** Database refreshes, regulatory updates
**Quarterly:** Comprehensive review and validation

### How to Verify Currency

1. **CFR Citations**: Cross-check with ecfr.gov for most recent version
2. **Guidance Documents**: Verify on FDA.gov (check for withdrawn/superseded)
3. **MDUFA/PDUFA Goals**: Confirm current user fee agreement in effect
4. **Contact Information**: Validate FDA phone/email on official site

### When Guidance Changes

- **New Guidance Published**: Review and integrate within 1 week
- **Guidance Withdrawn**: Flag immediately, update references
- **Regulation Amended**: Update citations, note effective dates
- **FDA Policy Shift**: Analyze implications, update recommendations

## Cross-Referencing FDA Official Sources

Always verify critical information at primary sources:

**Regulations:**
- 21 CFR: https://www.ecfr.gov (most current)
- Federal Register: https://www.federalregister.gov (proposed rules)

**Guidance:**
- FDA Guidance Documents: https://www.fda.gov/regulatory-information/search-fda-guidance-documents
- CDRH Guidance: https://www.fda.gov/medical-devices/device-advice-comprehensive-regulatory-assistance/guidance-documents-medical-devices-and-radiation-emitting-products

**Databases:**
- 510(k): https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfPMN/pmn.cfm
- PMA: https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfPMA/pma.cfm
- Drugs@FDA: https://www.accessdata.fda.gov/scripts/cder/daf/

**Standards:**
- FDA-Recognized Standards: https://www.accessdata.fda.gov/scripts/cdrh/cfdocs/cfStandards/search.cfm

## Tips for Effective Use

**Do:**
- Bookmark most-used reference sections
- Verify citations before critical decisions
- Cross-reference multiple sections for complete understanding
- Note dates on guidance documents
- Keep personal notes on interpretations

**Don't:**
- Assume old guidance is outdated (some remain current for decades)
- Skip verification of critical CFR citations
- Ignore footnotes and caveats
- Forget to check for updates quarterly
- Use secondary sources for final regulatory decisions
