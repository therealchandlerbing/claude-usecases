## Quick Reference Cards

### For Product Classification

**Device Classes:**
- **Class I**: Low risk, general controls (most exempt from 510(k))
- **Class II**: Moderate risk, special controls (usually 510(k))
- **Class III**: High risk, life-sustaining (PMA required)

**Quick Questions:**
```
What FDA class is [product]?
Intended use: [description]
Patient contact: [none/external/invasive/implant]
Technology: [how it works]
```

**Reference:** [FDA Regulations Reference](references/fda-regulations-reference.md)

### For Pathway Selection

**Medical Device Pathways:**
- **Exempt**: Class I devices, register and list only
- **510(k)**: Substantial equivalence, 3-6 months
- **De Novo**: Novel low-mod risk, 5-7 months
- **PMA**: Clinical data, 12-18 months

**Drug Pathways:**
- **IND**: Investigational, enables trials
- **NDA**: New drug, 10-12 months
- **ANDA**: Generic, 10-12 months
- **BLA**: Biologics, 10-12 months

**Reference:** [Submission Pathways Guide](references/submission-pathways-guide.md)

### For Compliance

**Quality System Essentials:**
- Design Controls (21 CFR 820.30)
- Document Control (21 CFR 820.40)
- CAPA (21 CFR 820.100)
- MDR (21 CFR 803)
- Complaints (21 CFR 820.198)

**GMP Essentials:**
- Process Validation
- Stability Programs
- Change Control
- Deviation Management
- Quality Metrics

**Reference:** [Quality Systems Reference](references/quality-systems-reference.md)

### For FDA Meetings

**Meeting Types:**
- **Q-Submission**: Agreement on testing, classification (90 days)
- **Pre-Submission**: Feedback on development plan (90 days)
- **Type A**: Critical path meetings (30 days)
- **Type B**: Milestone meetings (60 days)
- **Type C**: General guidance (75 days)

**Reference:** [FDA Meeting Request Template](templates/fda-meeting-request.md)

## Troubleshooting Guide

### "Where do I start?"

**New to FDA regulations:**
1. [QUICK-START.md](QUICK-START.md) - Get immediate guidance
2. [FDA Regulations Reference](references/fda-regulations-reference.md) - Learn basics

**Have specific question:**
1. Identify your category: Classification, Submission, Compliance, Strategy
2. Check relevant [Examples](examples/) directory
3. Ask question with sufficient context

**Need submission help:**
1. Identify pathway: 510(k), PMA, IND, NDA, etc.
2. Use corresponding [Template](templates/) checklist
3. Ask for specific guidance on gaps

### "Product classification unclear"

**Solutions:**
1. Review [Classification Examples](examples/classification-examples.md)
2. Check [FDA Regulations Reference](references/fda-regulations-reference.md) - Device classes
3. Ask: "What FDA class is [product description with intended use]?"
4. Consider FDA Q-Submission if still ambiguous

### "Not sure which pathway"

**Solutions:**
1. Read [Submission Pathways Guide](references/submission-pathways-guide.md)
2. Review [Submission Examples](examples/submission-examples.md)
3. Ask: "Which pathway for [product] given [current stage and context]?"
4. Compare options based on timeline, cost, and risk

### "Compliance question"

**Solutions:**
1. Check [Quality Systems Reference](references/quality-systems-reference.md)
2. Review [Compliance Examples](examples/compliance-examples.md)
3. Ask specific question about regulation or situation
4. Use [483 Response Template](templates/483-response-template.md) if inspection-related

### "FDA meeting preparation"

**Solutions:**
1. Review [FDA Meeting Request Template](templates/fda-meeting-request.md)
2. Check [Meeting Prep Examples](examples/meeting-prep-examples.md)
3. Ask: "How to prepare for [meeting type] about [topic]?"
4. Develop specific questions for FDA (agent can help)

### "Need submission checklist"

**Solutions:**
- Device clearance: [510(k) Checklist](templates/510k-checklist.md)
- Device approval: [PMA Checklist](templates/pma-checklist.md)
- Drug trials: [IND Checklist](templates/ind-checklist.md)
- Drug approval: [NDA Checklist](templates/nda-checklist.md)
