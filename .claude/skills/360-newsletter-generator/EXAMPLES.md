## Customization Patterns

### Adding Charts

```
Add a chart showing revenue by client
```

Response: Generates additional doughnut chart with client revenue breakdown.

### Expanding Sections

```
Expand the partnerships section with more detail
```

Response: Adds additional cards, more bullet points, deeper analysis.

### Changing Focus

```
Make this more strategic, less operational
```

Response: Emphasizes Strategic Horizon, reduces Operations detail.

### Adding Comparisons

```
Include comparison to last month
```

Response: Adds MoM comparison metrics and trend indicators.
