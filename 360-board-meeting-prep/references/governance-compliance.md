# Governance & Compliance Workflow
## Specialized Reference for Board Packet Governance Section

This file is loaded just-in-time when generating the Governance & Compliance Report component of the board packet.

---

## Purpose

Track board action items, compliance obligations, governance structure, and organizational housekeeping. This ensures nothing falls through the cracks and the board maintains proper oversight of 360 Social Impact Studios.

---

## Data Inputs

### From Asana Board/Governance Project

**Action Items**:
- Tasks assigned from prior board meetings
- Completion status and timeline
- Owners and deadlines
- Blockers or dependencies

**Compliance Calendar**:
- Filing deadlines (Form 990, state reports, etc.)
- Required document updates (conflict of interest, etc.)
- Annual requirements (officer elections, etc.)

### From Google Drive

**Prior Board Materials**:
- Prior meeting minutes
- Prior action item lists
- Governance documents (bylaws, policies)

### From User Context

**Board Composition**:
- Current board members
- Officer roles (Chair, Secretary, Treasurer)
- Term information
- Committee assignments (if applicable)

**Governance Calendar**:
- Meeting schedule for current year
- Any upcoming special meetings

## Data Collection Logic

### Asana Action Items

```python
def collect_governance_action_items():
    """Collect action items from Board/Governance Asana project"""

    board_project_gid = "1211794887178220"
    tasks = asana_get_tasks(project_gid=board_project_gid)

    action_items = []

    for task in tasks:
        task_detail = asana_get_task(task_gid=task['gid'])

        # Determine status
        if task_detail['completed']:
            status = '‚úÖ Complete'
            completion_date = task_detail['completed_at']
        elif task_detail.get('due_on') and task_detail['due_on'] < date.today():
            status = '‚ö†Ô∏è Delayed'
            completion_date = None
        elif 'blocked' in task_detail.get('notes', '').lower():
            status = 'üö´ Blocked'
            completion_date = None
        else:
            status = 'üîÑ In Progress'
            completion_date = None

        action_items.append({
            'description': task_detail['name'],
            'assigned_to': task_detail.get('assignee', {}).get('name', 'Unassigned'),
            'due_date': task_detail.get('due_on'),
            'status': status,
            'completion_date': completion_date,
            'notes': task_detail.get('notes', '')[:100]  # First 100 chars
        })

    return action_items
```

## Flag Generation Logic

```python
def generate_governance_flags(action_items, compliance_calendar):
    """Generate governance-related flags"""

    flags = []

    # Action item completion rate
    total_items = len(action_items)
    completed = len([i for i in action_items if i['status'] == '‚úÖ Complete'])
    completion_rate = (completed / total_items * 100) if total_items > 0 else 0

    if completion_rate < 75:
        flags.append({
            'severity': 'Warning',
            'category': 'Board Effectiveness',
            'message': f'‚ö†Ô∏è Only {completion_rate:.0f}% of action items completed',
            'recommendation': 'Review board capacity and action item prioritization'
        })

    # Overdue action items
    delayed = [i for i in action_items if i['status'] == '‚ö†Ô∏è Delayed']
    if delayed:
        flags.append({
            'severity': 'Info',
            'category': 'Action Items',
            'message': f'‚ÑπÔ∏è {len(delayed)} action items overdue',
            'recommendation': f'Follow up on: {", ".join([i["description"][:30] for i in delayed[:3]])}'
        })

    # Compliance deadlines
    for item in compliance_calendar:
        if item['status_icon'] == '‚ö†Ô∏è':
            flags.append({
                'severity': 'Critical',
                'category': 'Compliance',
                'message': f'üö® {item["requirement"]} is overdue (due {item["due_date"]})',
                'recommendation': 'Immediate action required to avoid compliance lapse'
            })
        elif item['status_icon'] == '‚è∞':
            flags.append({
                'severity': 'Info',
                'category': 'Compliance',
                'message': f'‚ÑπÔ∏è {item["requirement"]} due {item["due_date"]}',
                'recommendation': 'Ensure responsible party is aware and on track'
            })

    return flags
```
