## Quick Navigation

### For First-Time Users
1. Read [README.md](README.md) - Overview and quick start
2. Review [prompt-templates.md](references/prompt-templates.md) - Pick a template
3. Paste template into Claude (with SKILL.md loaded in Project)
4. Review output with [quality-checklist.md](references/quality-checklist.md)

### For Strategic Planning
1. Read [platform-strategy.md](references/platform-strategy.md) - Choose platforms strategically
2. Use Template 6 or Template 7 from [prompt-templates.md](references/prompt-templates.md)
3. Review strategic alignment in [quality-checklist.md](references/quality-checklist.md)

### For Regular Users
1. Pick appropriate template from [prompt-templates.md](references/prompt-templates.md)
2. Customize with your content and requirements
3. Review output quality with [quality-checklist.md](references/quality-checklist.md)
4. Export and publish

### For Team Leaders
1. Share [system-guide.md](references/system-guide.md) with team
2. Create team-specific templates (customize from [prompt-templates.md](references/prompt-templates.md))
3. Assign roles using [quality-checklist.md](references/quality-checklist.md) role-specific sections
4. Set up approval workflow

## Template Categories (prompt-templates.md)

1. **Basic Conversions** - Simple multi-platform, quick social, professional formats
2. **Multi-Language Conversions** - Multi-language multi-platform, single language adaptation
3. **Strategic Conversions** - Platform strategy first, ecosystem thinking, thought leadership
4. **Audience-Specific** - Multiple segments, technical to general
5. **Format-Specific** - Email sequences, presentations, podcasts, infographics
6. **Design-Ready** - Complete design specification packages
7. **Testing & Optimization** - A/B variations, analytics-driven improvements

## Quality Check Categories (quality-checklist.md)

1. **Critical Issues** - Accuracy, brand voice, basic quality, sensitive content
2. **Platform-Specific** - LinkedIn, Twitter, Email, Instagram, Blog, Presentations, Podcasts, Infographics
3. **Multi-Language** - Translation quality, cultural appropriateness, technical checks
4. **Design-Ready** - Specifications, accessibility, handoff readiness
5. **Strategic Alignment** - Goals, audience fit, timing, resource capacity
6. **Delegation** - Team handoff, specifications, workflow

## Platform Strategy Categories (platform-strategy.md)

1. **Content Type Pathways** - Long-form, quick insights, data/research, stories, announcements, educational
2. **Goal-Based Pathways** - Thought leadership, reach, engagement, lead generation
3. **Audience-Based** - B2B, B2C, technical experts, executives
4. **Resource-Based** - Low, medium, high capacity recommendations
5. **Platform Synergies** - Which platforms work well together
6. **Testing & Optimization** - How to test and improve over time
