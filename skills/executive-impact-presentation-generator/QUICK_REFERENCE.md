# Executive Impact Presentation Generator - Quick Reference

## 5-Minute Start Guide

### Step 1: Trigger the Skill
Say: "Create an executive impact report for [organization name]"

### Step 2: Provide Essentials
**Minimum required:**
- Organization name
- Brand color (hex code, e.g., #FF6B35)
- Fiscal year
- Key metrics for 6 sections

### Step 3: Get Both Formats
**Outputs:**
- Presentation format (landscape slides for meetings)
- Executive format (portrait pages for sharing)

### Step 4: Export to PDF
**In Chrome:**
- Cmd+P (Mac) or Ctrl+P (Windows)
- Presentation → Landscape
- Executive → Portrait
- Save as PDF

---

## What You Need to Provide

### Required Information

| Section | What You Need |
|---------|--------------|
| **Organization** | Name, brand color, fiscal year |
| **Impact Overview** | 3 reach metrics, 3-5 momentum highlights |
| **Regional Portfolio** | 6 region allocations (name, %, description), 3 priorities |
| **Financial Performance** | 4-5 capital metrics, 3-part capital mix, 2 safeguards |
| **Program Outcomes** | 3 impact pillars (each with metric + 3 outcomes), 3 evaluation principles |
| **Partnerships** | 3 overall stats, 4 partner categories, 4 flagship partnerships |
| **Strategic Outlook** | 3 horizon roadmap, 3-4 leadership asks |

## Content Guidelines

**Metrics:**
- Use K for thousands: 847K
- Use M for millions: $42M
- Include comparison: +41% from FY 2024
- Round to whole numbers

**Descriptions:**
- Under 20 words each
- Active voice, present tense
- Specific numbers and dates
- Front-load key information

**Regional Percentages:**
- Must total 100%
- 6 regions required
- Can include "Global Initiatives"

**Capital Mix:**
- Must total 100%
- 3 categories required
- Reflects deployment, not commitments

## Troubleshooting Quick Fixes

**Content overflows presentation slide**
→ Reduce text length or move to Executive format only

**Brand color not showing**
→ Verify hex code format (#RRGGBB)

**Navigation broken**
→ Check slide numbers are sequential (0-6)

**Print layout wrong**
→ Use Chrome, set correct orientation, enable background graphics

**Content misaligned**
→ Verify all required schema fields populated

## Export Instructions

### Presentation Format (Landscape Deck)
1. Open HTML in Chrome
2. Press Cmd+P (Mac) or Ctrl+P (Windows)
3. Destination: "Save as PDF"
4. Layout: **Landscape**
5. Margins: None
6. Options: ✓ Background graphics
7. Save with descriptive name

**Result:** One slide per page, deck format

### Executive Format (Portrait Report)
1. Open HTML in Chrome
2. Press Cmd+P (Mac) or Ctrl+P (Windows)
3. Destination: "Save as PDF"
4. Layout: **Portrait**
5. Options: ✓ Background graphics
6. Save with descriptive name

**Result:** Multi-page document, continuous flow

## Usage Patterns

### Use Presentation Format When:
- Presenting live to board
- Need deck for meeting
- One-page-per-topic layout preferred
- Creating slide-based materials

### Use Executive Format When:
- Emailing to stakeholders
- Comprehensive review needed
- Multi-page sections acceptable
- Detailed reading expected

### Generate Both When:
- Different audiences (meeting vs email)
- Want flexibility in distribution
- Creating full documentation package
- Archiving annual results
