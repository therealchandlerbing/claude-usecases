## Common Patterns Across Examples

### Intelligence Gathering
All campaigns start with research:
- Past conversations
- Email history
- Meeting history
- Web search (for new prospects)
- Recent news and developments

### Email Structure
Every email follows this pattern:
- **Subject**: Clear value or curiosity
- **Hook**: Relevant insight or connection
- **Context**: Show you understand their world
- **Value**: What you offer
- **Social Proof**: Case study or result
- **CTA**: One clear next step

### Personalization
All emails include:
- First name
- Company name
- Specific initiatives or challenges
- Recent developments
- Relevant case studies

### Deal Tracking
All campaigns create Asana tasks with:
- Custom fields (stage, value, close date)
- Subtasks for follow-ups
- Comments logging actions
- Next action defined

## Example Variations

### By Prospect Type

**Cold Prospects** (No prior relationship):
- Longer sequences (5+ touches)
- Slower cadence (3+ days between emails)
- More value education needed
- Conversion: 5-15%

**Warm Prospects** (Existing relationship):
- Shorter sequences (3 touches)
- Faster cadence (2-3 days between emails)
- Reference past conversations
- Conversion: 20-40%

**Stuck Deals** (Gone silent):
- 3-touch sequences
- Acknowledge gap directly
- Provide new information/urgency
- Offer alternative paths

### By Industry

**B2B SaaS**:
- Focus on scaling metrics
- Reference ARR, response rates, deal velocity
- Case studies with similar stage companies

**Consulting Services**:
- Emphasize expertise and process
- Reference client outcomes and ROI
- Case studies with similar challenges

**Agency Partnerships**:
- Focus on collaboration and mutual benefit
- Reference portfolio and past projects
- Case studies with successful partnerships

### By Geographic Region

**US Corporate**:
- Direct, concise emails
- Lead with data and ROI
- Fast cadence

**Brazilian Partners**:
- Warmer tone, relationship-first
- More context before ask
- Portuguese option when appropriate

**Asian Partners**:
- Formal approach
- Group harmony emphasis
- Indirect communication style
