# Intelligence Extraction Template Selection Guide

Quick reference for choosing the right extraction template based on meeting type and context.

## Quick Decision Tree

```
New organization, first meeting?
├─ Partnership focus → Template 1 (New Partnership Exploration)
├─ Funding focus → Template 3 (Funder Initial Contact)
└─ Individual focus → Check their role...
   ├─ Board member → Template 5 (Board Meeting)
   ├─ Community leader → Template 7 (Community Stakeholder)
   └─ Other → Template 9 (Networking) or Template 1

Existing relationship, check-in meeting?
├─ Partnership → Template 2 (Existing Partnership Check-in)
├─ Funder → Template 4 (Grant Application Discussion)
├─ Client → Template 6 (Client Sprint)
└─ Board → Template 5 (Board Meeting)

Working session or facilitation?
└─ Template 6 (Client Sprint)

Problem or crisis?
└─ Template 10 (Crisis/Problem-Solving)

International/cross-cultural context?
└─ Template 8 (International Partner)

Community engagement?
└─ Template 7 (Community Stakeholder)

Brief conference interaction?
└─ Template 9 (Conference Networking)

Multiple organizations present?
└─ Use Advanced Template (Multi-Entity)
```

## Template Details

### Template 1: New Partnership Exploration
**File:** `01-partnership-new.md`

**Use when:**
- First or second meeting with potential partner organization
- Exploring collaboration possibilities
- Initial relationship building

**Extracts:**
- Organization problems and urgency
- Cultural communication patterns
- Fit assessment signals
- Decision process intelligence
- Next steps clarity

**Time required:** 15-20 minutes

---

### Template 2: Existing Partnership Check-in

## Special Situations

### Multiple Meeting Types
**Example:** Emergency board meeting to discuss partnership crisis

**Solution:**
1. Primary template: Template 10 (Crisis)
2. Reference Template 5 guidance for board dynamics
3. Note in context which lenses you're using

### Unclear Meeting Type
**When in doubt:**
- Start with **Template 1** (most general)
- Adjust mid-extraction if another template fits better
- Flag for review and note which template you should have used

### Multi-Stakeholder Events
**Example:** Coalition meeting with 5 organizations

**Solution:**
- Use **Advanced Template: Multi-Entity Extraction**
- Extract Partnership Intelligence for each relevant org
- Extract Stakeholder Intelligence for key individuals
- Note ecosystem dynamics

### Follow-up to Crisis
**After crisis is resolved:**
- Use Template 2 (Existing Partnership) if partner
- Include notes on crisis recovery in relationship evolution
- Track whether crisis changed dynamics permanently

## Customization Tips

### Context Matters
Always fill in the context section:
- **Organization name and type**
- **Meeting stage** (first call, ongoing, renewal, etc.)
- **Cultural context** (especially for international)
- **360 participants and partner participants**

### When to Flag
Flag for review if:
- Unsure which template to use
- Meeting covered multiple contexts
- Transcript quality was poor
- Critical information seems to be missing

### Template Combinations
You can reference multiple templates:
1. Start with primary template
2. Add "Also consider [Template X]'s focus on [specific aspect]"
3. Note in output which lenses were combined
