2. **Spreadsheets** - Excel, Google Sheets
3. **HTML/Web Interfaces** - Static sites, dashboards, web apps
4. **Reports/Documents** - Word, Google Docs, Markdown
5. **Dashboards** - Data visualization, analytics interfaces

### Key Exemplars
- **Stripe** - Financial UI, data-heavy interfaces, trust
- **Linear** - Product interfaces, modern SaaS, efficiency
- **Apple** - Premium positioning, elegance, restraint
- **Notion** - Content-first, collaborative, readable
- **Bauhaus** - Geometric, systematic, bold
- **Swiss Design** - Typographic excellence, grids, clarity
- **Brutalism** - Raw, developer-focused, unconventional

## Skill Workflow

```
1. User provides functional output + context
   ↓
2. Skill evaluates against Design Interrogation Checklist
   ↓
3. Skill selects appropriate exemplar(s)
   ↓
4. Skill chooses 2-3 high-impact techniques
   ↓
5. Skill applies techniques systematically
   ↓
6. Skill validates against quality standards
   ↓
7. User receives polished output (process typically invisible)
```
