# 360 Executive Brief Documentation

## Overview

This directory contains comprehensive documentation for creating and maintaining the 360 Impact Brief - an executive intelligence dashboard that synthesizes partnerships, operations, strategy, and finance into actionable weekly insights.

## Primary Use Case

The **360 Executive Brief** is the primary output format for 360 Social Impact Studios. It provides:

- **Confidential executive intelligence** for leadership, board, and investors
- **Financial metrics and projections** with strategic context
- **HR and operational updates** including hiring and capacity
- **Partnership pipeline intelligence** with deal stages and values
- **Strategic risk assessment** with mitigation strategies
- **Forward-looking timeline** with critical dates and decisions

## What You'll Find Here

### Complete Workflow System

This documentation provides a complete workflow for:

1. **Creating weekly briefs** from scratch
2. **Updating existing briefs** with new data
3. **Maintaining consistency** across all sections
4. **Ensuring quality** before distribution
5. **Troubleshooting issues** when they arise

### Content Governance

Strong governance rules ensure every brief is:

- **Concise**: Executive snapshot cards limited to 2 sentences
- **Focused**: Each story lives in one section only
- **Actionable**: Every metric has context and interpretation
- **Consistent**: Fixed structure week over week
- **Professional**: WCAG AA accessible, print-ready

### Technical Excellence

The system provides:

- **Interactive timeline** with toggle views (This Week / Next 2 Weeks)
- **Dynamic charts** with Chart.js (revenue, capacity, market fit)
- **Collapsible sections** with persistent state
- **Smooth navigation** with scroll-to-section
- **Mobile responsive** design that works on all devices
- **Print optimization** for board meetings and reviews

## Design Philosophy

### Executive-First

Every design decision prioritizes executive needs:
- **Scannable structure** - Key info in fixed locations
- **Decision-focused** - Lead with what needs action
- **Context always** - Every metric interpreted
- **Visual hierarchy** - Most important info most prominent

### Governed Content

Strong rules prevent content bloat:
- **Word limits** strictly enforced
- **Section ownership** - Each story in one place
- **Cross-reference** instead of repeating
- **Impact before detail** always

### Technical Excellence

Production-quality implementation:
- **Accessible** - WCAG AA compliant
- **Responsive** - Works on all devices
- **Print-ready** - Clean output for meetings
- **Performant** - Fast load, smooth interactions
